Capitol Words API

dates.json

Required arguments

  • phrase

    The phrase to search for.

Optional arguments

  • entity_type

    The entity type.

    Valid values:

    • state
    • party
    • legislator
    • bioguide

    If this argument is supplied, entity_value is required.

  • entity_value

    The value of the entity given in entity_type.

    If this argument is supplied, entity_type is required.

  • start_date

    Limit results to those starting at the given date.

    If this argument is supplied, end_date is required.

  • end_date

    Limit results to those ending at the given date.

    If this argument is supplied, start_date is required.

  • granularity

    The length of time covered by each result

    Valid values:

    • year
    • month
    • week
  • page

    The page of results to show.

    100 results are shown at a time. To get more than 100 results, use the page parameter.

Examples

  • dates.json?phrase=united+states
  • dates.json?phrase=united+states&entity_type=state&entity_value=VA
  • dates.json?phrase=united+states&start_date=2010-01-01&end_date=2010-06-01
  • dates.json?phrase=united+states&start_date=2010-01-01&end_date=2010-06-01&granularity=month

phrases/{entity_type}.json

{entity_type} must be one of:

  • legislator
  • state
  • party

Required arguments

  • phrase

    The phrase to search for.

Optional arguments

  • start_date

    Limit results to those starting at the given date.

    If this argument is supplied, end_date is required.

  • end_date

    Limit results to those ending at the given date.

    If this argument is supplied, start_date is required.

  • sort

    By default, results are sorted alphabetically.

    To sort in descending order by the number of hits per entity type, set to "true".

Examples

  • /phrases/legislator.json?phrase=obama+administration
  • /phrases/legislator.json?phrase=obama+administration&sort=true
  • /phrases/legislator.json?phrase=obama+administration&start_date=2010-01-01&end_date=2010-06-01

phrases.json

List the top phrases.

Optional arguments

  • n

    The size of phrase, in words, to search for (up to 5).

  • start_date

    Limit results to those starting at the given date.

    If this argument is supplied, end_date is required.

  • end_date

    Limit results to those ending at the given date.

    If this argument is supplied, start_date is required.

  • entity_type

    The entity type.

    Valid values:

    • state
    • party
    • legislator
    • bioguide

    If this argument is supplied, entity_value is required.

  • entity_value

    The value of the entity given in entity_type.

    If this argument is supplied, entity_type is required.

  • page

    The page of results to show.

    100 results are shown at a time. To get more than 100 results, use the page parameter.

  • stemmed

    Set as 'true' to use stemming in top-word list.

    Note that this works only if n=1

Examples

  • /phrases.json
  • /phrases.json?top=50
  • /phrases.json?top=50&n=3
  • /phrases.json?start_date=2010-01-01&end_date=2010-06-01
  • /phrases.json?top=50&n=3&entity_type=party&entity_name=Democrat

legislators.json

Look up a legislator by Bioguide ID

Required arguments

  • bioguide_id

    The Bioguide ID of the legislator to look up.

Examples

  • /legislators.json?bioguide_id=R000053
